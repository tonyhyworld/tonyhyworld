# 张诺亚 / Tony Zhang

I build **AI World / TraceArena** — the open-source runtime for auditable multi-agent worlds: real tools, enforceable rules, verifiable outcomes, and watchable runs.

## What I’m building

Multiple agents act inside a shared world. The world validates evidence, structured actions, events, settlement, and replay. Competition turns decisions and consequences into auditable data for agent evaluation and training.
