# 张诺亚 / Tony Zhang

I build **AI World / TraceArena** — the open-source **Physical World OS** for multi-agent decision worlds.

## What I’m building

The goal is to make physical-world problems runnable: load resources, goals, rules, tools, information boundaries, and authoritative outcomes into a world contract. Multiple agents can then compete or cooperate under shared constraints; the world simulates consequences, settles outcomes, and compares paths under the declared objective and evidence.

This is not a training-data repository or a chatbot wrapper. Replayable traces are evidence for inspecting and improving decisions. The long-term vision is one extensible OS where operations, logistics, governance, incident response, research, and market problems can be loaded as scenario packs.
